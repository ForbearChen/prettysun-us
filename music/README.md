# 🎵 音乐文件夹

## 如何添加背景音乐

1. 准备一个音乐文件（支持 MP3、WAV、OGG 格式）
2. 将文件重命名为 `song.mp3`
3. 将文件放到这个 `music` 文件夹中
4. 重新访问网站，点击右下角的音符图标即可播放

## 推荐设置

- **文件格式**: MP3（兼容性最好）
- **文件大小**: 建议不超过 10MB，以保证加载速度
- **音质**: 128-192 kbps 即可，音质与文件大小的平衡
- **时长**: 3-5 分钟的歌曲比较合适

## 版权提醒

⚠️ 请确保你使用的音乐文件不侵犯版权。推荐使用：
- 你自己创作的音乐
- 免费版权音乐网站的资源
- 已获得授权的音乐

## 故障排除

如果音乐无法播放：
1. 确认文件名正确：`song.mp3`
2. 确认文件路径正确：`music/song.mp3`
3. 确认文件格式正确
4. 在浏览器中按 F12 打开开发者工具，查看控制台是否有错误信息

## 提示

💡 你也可以使用其他文件名，但需要修改 `index.html` 和其他 HTML 文件中的以下代码：

```html
<source src="music/song.mp3" type="audio/mpeg">
```

将 `song.mp3` 改为你的文件名。
