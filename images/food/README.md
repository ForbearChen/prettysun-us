# 🍜 Food - 美食照片

## 用途
存放你们一起吃饭、做饭、品尝美食的照片。

## 照片示例
- 街边小吃的合照
- 精致餐厅的美食
- 节日聚餐的场景
- 一起下厨做饭的照片
- 咖啡厅下午茶的照片

## 命名示例
- `2018_12_25_christmas_dinner_01.jpg` - 圣诞晚餐
- `2019_02_14_valentine_dessert_01.jpg` - 情人节甜点
- `2020_08_15_mooncake_festival_01.jpg` - 中秋月饼
- `2021_05_20_anniversary_cake_01.jpg` - 纪念日蛋糕
