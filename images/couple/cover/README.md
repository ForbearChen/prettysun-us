# 💕 Couple Cover - 情侣章节封面

## 封面图片数量：2张

这些图片将以堆叠布局显示在故事页面的情侣章节中。

## 建议使用的图片类型
- 重要纪念日的合影
- 浪漫的约会场景
- 温馨的日常生活合照
- 节日庆祝的甜蜜瞬间

## 当前使用的图片
```
2018_12_06_couple_01.jpg  # 情侣合影
2018_12_31_couple_01.jpg  # 跨年时刻
```

## 特点
这2张图片代表了情侣关系中的重要时刻：
- 日常的情侣合影
- 特殊节日（跨年）的温馨时刻

## 命名要求
- 放在 `images/couple/cover/` 文件夹中
- 保持原文件名不变或重命名为 `couple_cover_01.jpg` 等格式
- 支持 JPG/PNG/WebP 格式

## 建议
如果有更多温馨的情侣照片，也可以考虑增加到3张。
