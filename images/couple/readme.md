# 💕 Couple - 情侣合照

## 用途
存放你们作为情侣的重要纪念时刻照片。

## 照片示例
- 重要纪念日的合照
- 节日庆祝的照片
- 特殊场合的正式合照
- 浪漫的约会场景
- 一起成长的见证照片

## 命名示例
- `2018_05_20_first_anniversary_01.jpg` - 一周年纪念
- `2019_09_10_teachers_day_01.jpg` - 教师节惊喜
- `2020_12_31_newyear_eve_01.jpg` - 跨年夜拥抱
- `2021_11_11_singles_to_couple_01.jpg` - 从光棍到情侣

## 故事章节
这些照片将用于 **第五章：彼此依靠**
