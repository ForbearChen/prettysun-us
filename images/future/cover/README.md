# 🌟 Future Cover - 未来章节封面

## 封面图片数量：1张

这张图片将以特殊布局显示在故事页面的未来章节中，作为该章节的代表性图片。

## 建议使用的图片类型
- 婚纱摄影或婚礼相关
- 毕业典礼场景
- 关于未来的想象照片
- 职业规划或梦想相关的图片

## 当前使用的图片
```
2018_11_12_cherish_01.jpg  # 珍惜当下
```

## 特点
这张图片以单张大图的形式展示，突出了"珍惜当下，展望未来"的主题。

## 命名要求
- 放在 `images/future/cover/` 文件夹中
- 文件名格式：`future_cover_01.jpg`
- 支持 JPG/PNG/WebP 格式

## 建议
如果将来有更多关于未来的照片（如婚纱照、毕业照等），可以考虑增加到2-3张，形成一个小系列。
