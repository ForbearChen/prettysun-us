# 🌟 Future - 关于未来的照片

## 用途
存放关于你们未来规划和憧憬的照片。

## 照片示例
- 婚纱摄影或婚礼相关
- 毕业典礼的预演照片
- 未来家庭的想象照片
- 梦想中的旅行目的地
- 职业规划相关的照片

## 命名示例
- `2025_01_01_wedding_dress_01.jpg` - 婚纱照预览
- `2026_06_01_graduation_ceremony_01.jpg` - 毕业典礼
- `2030_01_01_family_portrait_01.jpg` - 未来全家福
- `2028_07_15_world_trip_01.jpg` - 世界旅行计划

## 故事章节
这些照片将用于 **终章：未来可期**
