## 📝 照片命名规范

### **推荐命名格式：**
```
[年份]_[月份][日期]_[描述]_[序号].[格式]
```

### **示例：**
- `2018_03_15_library_corner_01.jpg` - 2018年3月15日图书馆角落
- `2019_02_14_valentine_dessert_01.jpg` - 2019年情人节甜点
- `2020_08_15_mooncake_festival_01.jpg` - 2020年中秋月饼

### **命名规则：**
- **日期格式**: `YYYY_MM_DD` (如: `2018_03_15`)
- **描述**: 英文小写 + 下划线分隔，简洁明了
- **序号**: 从 `01` 开始编号，同主题多张照片用序号区分
- **格式**: 建议使用 `.jpg` 或 `.png`

## 📏 照片规格建议

- **文件大小**: 每张不超过 2MB
- **分辨率**: 至少 1500x1125 像素
- **格式**: JPG/PNG/WebP
- **宽高比**: 根据使用位置调整（故事页面建议 3:4 或 4:3）

## 🔧 使用方法

将照片放入对应文件夹后，需要更新 `story.html` 中的图片路径：

```html
<!-- 替换占位图片 -->
<img src="images/study/2018_03_15_library_corner_01.jpg" alt="图书馆角落">
```

## 🎥 Live Photo 支持

网站现在支持 **Live Photo** 功能！你可以将 iPhone 的 Live Photo（MOV格式）上传到对应文件夹中：

### 如何添加 Live Photo

1. **导出MOV文件**: 在 iPhone 相册中选择 Live Photo，点击分享 → 导出 → 选择 MOV 格式
2. **命名文件**: 使用与静态图片相同的文件名，但扩展名为 `.mov`
3. **放置位置**: 放在与对应静态图片相同的文件夹中

### 示例文件结构
```
images/study/
├── 2018_11_07_teacherhuang_01.jpg    # 静态图片
├── 2018_11_07_teacherhuang_01.mov    # Live Photo (可选)
└── ...
```

### Live Photo 特性

- 🖱️ **桌面端**: 鼠标悬停自动播放，点击播放按钮可手动播放
- 📱 **移动端**: 点击图片播放，播放完成后自动停止
- 🎬 **模态框**: 在全屏画廊中查看时可控制播放进度
- ⚡ **性能优化**: 视频按需加载，支持预加载控制

## 🎭 Easter Eggs 彩蛋照片

每个主题文件夹都包含一个 `easter-eggs/` 子文件夹，用于存放：

- **搞笑瞬间**: 学习、生活中的趣事
- **可爱表情**: 宠物捣蛋、搞怪照片
- **温馨趣事**: 情侣间的甜蜜搞笑时刻
- **隐藏彩蛋**: 只有在特定情况下才会显示

## 🎯 使用建议

### Cover 文件夹
- 精选最具代表性的照片
- 按照指定数量放置
- 这些照片会在网站主要展示区域显示

### 主文件夹
- 存放所有其他照片
- 数量不限，可以不断添加
- 通过"查看更多"功能展示

### Easter Eggs 文件夹
- 存放搞笑、有趣的照片
- 可以作为隐藏内容或彩蛋
- 为网站增添更多乐趣

## 💡 提示

- 照片文件名要具有描述性，便于管理和维护
- 定期备份重要照片
- 确保照片内容适合公开分享
- 彩蛋照片要确保积极正面，避免尴尬内容
- Live Photo 文件大小建议控制在 50MB 以内
