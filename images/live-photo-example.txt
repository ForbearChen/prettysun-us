# Live Photo 使用示例

## 📸 如何制作和添加 Live Photo

### 1. 从 iPhone 导出 Live Photo

1. 打开「照片」App
2. 选择包含 Live Photo 的照片（照片右上角会有「Live」标识）
3. 点击分享按钮（方形带箭头图标）
4. 选择「导出」
5. 选择「MOV」格式
6. 保存到电脑

### 2. 文件命名规则

Live Photo 文件名必须与对应的静态图片完全相同，只是扩展名不同：

```
静态图片: 2018_12_25_christmas_dinner_01.jpg
Live Photo: 2018_12_25_christmas_dinner_01.mov
```

### 3. 放置位置

将 MOV 文件放在与静态图片相同的文件夹中：

```
images/couple/
├── 2018_12_25_christmas_dinner_01.jpg
└── 2018_12_25_christmas_dinner_01.mov  ← Live Photo
```

### 4. 网站功能

- **自动检测**: 网站会自动检测是否有对应的 MOV 文件
- **播放控制**:
  - 桌面端：鼠标悬停播放，点击播放按钮
  - 移动端：点击图片播放
- **全屏查看**: 在「查看更多」模态框中可以控制视频播放

### 5. 技术说明

- 支持 H.264 编码的 MOV 文件
- 文件大小建议不超过 50MB
- 会自动生成播放控制按钮和 Live 标识
- 支持视频预加载优化

## 🎯 提示

如果你的 Live Photo 没有正确显示，请检查：
1. 文件名是否与静态图片完全匹配
2. 文件格式是否为 MOV
3. 文件是否损坏
4. 浏览器是否支持视频播放
