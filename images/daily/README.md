# 🏠 Daily - 日常生活照片

## 用途
存放你们日常生活中的温馨瞬间。

## 照片示例
- 周末宅在家里的照片
- 一起购物采购的场景
- 宠物或家庭成员的合照
- 节日庆祝的照片
- 日常小确幸的瞬间

## 命名示例
- `2018_11_11_singles_day_cooking_01.jpg` - 光棍节做饭
- `2019_04_20_cherry_blossom_01.jpg` - 樱花季出游
- `2020_06_01_childrens_day_01.jpg` - 儿童节庆祝
- `2021_12_31_newyear_countdown_01.jpg` - 跨年倒计时

## 故事章节
这些照片将用于 **第四章：温馨日常**
