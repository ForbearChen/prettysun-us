# 🏠 Daily Cover - 日常生活章节封面

## 封面图片数量：4张

这些图片将以网格布局显示在故事页面的日常生活章节中。

## 建议使用的图片类型
- 温馨的居家场景
- 节日庆祝瞬间
- 日常生活的小确幸
- 宠物或家庭成员的合影

## 特点
这4张图片已经在网站上正常显示，代表了日常生活中的不同场景：
- 日常生活场景
- 个人写真
- 娱乐活动
- 数字生活记录

## 命名要求
- 放在 `images/daily/cover/` 文件夹中
- 保持原文件名不变或重命名为 `daily_cover_01.jpg` 等格式
- 支持 JPG/PNG/WebP 格式
