# ✈️ Travel - 旅行照片

## 用途
存放你们一起旅行、出游的照片。

## 照片示例
- 景点合照
- 旅行途中的风景
- 酒店或民宿的照片
- 当地特色的美食
- 旅行纪念品或小物

## 命名示例
- `2019_05_01_beijing_wall_01.jpg` - 五一北京长城
- `2019_07_15_sanya_beach_01.jpg` - 三亚沙滩游
- `2021_10_01_everest_base_01.jpg` - 珠峰大本营
- `2022_03_15_japan_cherry_01.jpg` - 日本樱花季
