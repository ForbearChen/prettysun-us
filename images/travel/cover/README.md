# ✈️ Travel Cover - 旅行章节封面

## 封面图片数量：3张

这些图片将以堆叠布局显示在故事页面的旅行章节中。

## 建议使用的图片类型
- 标志性景点或地标
- 旅行中的精彩瞬间
- 当地文化特色场景
- 自然风景或城市风光

## 当前使用的图片
```
2018_09_30_jaychou_01.jpg  # 周杰伦演唱会
2018_09_30_jaychou_02.jpg  # 演唱会现场
```

## 补充建议
需要再添加1张旅行照片来达到3张封面的要求。可以选择：
- 其他演唱会场景
- 旅行途中的风景
- 户外活动照片

## 命名要求
- 放在 `images/travel/cover/` 文件夹中
- 文件名格式：`travel_cover_01.jpg`, `travel_cover_02.jpg`, `travel_cover_03.jpg`
- 支持 JPG/PNG/WebP 格式
