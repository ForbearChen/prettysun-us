# 📚 Study - 一起读书的照片

## 用途
存放你们一起学习、读书、学术相关的照片。

## 照片示例
- 图书馆学习的场景
- 一起复习备考的照片
- 论文写作时的合照
- 学术会议或讲座的照片
- 毕业论文答辩的纪念照

## 命名示例
- `2018_09_01_freshman_registration_01.jpg` - 大一开学注册
- `2019_06_12_library_midnight_01.jpg` - 图书馆通宵复习
- `2020_03_15_thesis_defense_01.jpg` - 论文答辩
- `2021_07_01_graduation_ceremony_01.jpg` - 毕业典礼
