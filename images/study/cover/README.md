# 📚 Study Cover - 学习章节封面

## 封面图片数量：3张

这些图片将显示在故事页面的学习章节中，作为该章节的代表性图片。

## 建议使用的图片类型
- 与老师或导师的合影
- 学习时的专注瞬间
- 学术成果或证书照片
- 图书馆或实验室场景

## 当前使用的图片
```
2018_11_07_teacherhuang_01.jpg  # 黄老师指导
2018_11_28_teacherhuang_01.jpg  # 黄老师论文指导
2019_01_04_teacherhuang_01.jpg  # 学术讨论
```

## 命名要求
- 放在 `images/study/cover/` 文件夹中
- 保持原文件名不变
- 支持 JPG/PNG/WebP 格式
